Maxsus Modal va Ogohlantirishlar Demosi

Tasdiqlash Modallari

Brauzerning standart confirm() funksiyasini almashtiradigan har xil turdagi tasdiqlash dialoglarini sinab ko'ring.

Bildirishnomalar

Har xil uslub va davomiylikdagi bildirishnomalarni sinab ko'ring.

Bootstrap Ogohlantirishlarini O'zgartirish

An'anaviy Bootstrap ogohlantirishlari sahifa yuklanganda avtomatik ravishda maxsus bildirishnomalarga aylantiriladi.

Bu muvaffaqiyatli Bootstrap alert - avtomatik toast bo'lib aylanadi!
Bu xatolik Bootstrap alert - toast shaklida ko'rsatiladi!
Bu ogohlantirish Bootstrap alert - toast formatida chiqadi!

Tasdiqlash Modali
// Oddiy foydalanish
customConfirm('Xabar?', (confirmed) => {
  if (confirmed) {
    // Foydalanuvchi "Ha" ni bosdi
  }
});

// Murakkab foydalanish
customConfirmAdvanced({
  title: 'Maxsus Sarlavha',
  message: 'Sizning xabaringiz',
  confirmText: 'O\'chirish',
  cancelText: 'Bekor qilish',
  confirmClass: 'btn-danger',
  callback: (confirmed) => { ... }
});
